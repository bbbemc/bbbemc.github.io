ยินดีต้อนรับส่วนของเนื้อหาครับ !!

ผมสนใจในด้านการทำเว็บ AI เเละ เทคโนโลยีใหม่ๆ
ภาษาโปรแกรมภาษาแรกที่ผมได้เรียนคือ Python ซึ่งได้เรียนรู้จากการเรียนการสอนในโรงเรียน หลังจากที่ผมได้ลองเขียนภาษาโปรเเกรมครั้งเเรก ผมสงสัยมากๆเกี่ยวกับภาษาโปรเเกรม เลยได้ไปศึกษาเกี่ยวกับภาษาโปรเเกรมมากขึ้น
ว่า มันทำอะไรได้บ้าง หลักการมันเป็นยังไง หลังจากที่ได้ศึกษาเเละอ่านมันเป็นอะไรที่เปิดโลกผมมากเลยครับ ภาษาโปรเเกรมมีหลายเเบบเเละทำได้หลายอย่างมากๆ
จากจุดนั้นทำให้ผมเริ่มศึกษามาเรื่อยๆโดยเฉพาะ ภาษา HTML css เเละ Javascript ที่ต้องใช้ด้วยกัน
เพราะ เราสามารถทำสิ่งที่เราคิดเเละจินตนาการไว้ในหัว ให้เกิดขึ้นเป็นจริงได้ ผ่านการเขียนโปรเเกรม ผมเป็นอีกคนที่วาดรูปไม่เก่งเเต่อยากมีผลงานสวยๆเป็นของตัวเองบ้าง
ซึ่งเว็บหรือ ภาษาโปรเเกรมเป็นตัวเลือกนึงที่ผมสนใจมากๆ เพราะ มันทำให้ผมออกเเบบเองได้ในเเบบที่ผมต้องการ ซึ่งการเรียนรู้ไม่ใช่ปัญหาเลย
เพราะ การที่คนเราจะเก่งได้ ต้องมีการเรียนรู้เเละคิดอยากจะลงมือทำสิ่งนั้น ชอบในสิ่งนั้นจริงๆ ไม่มีข้ออ้างหรือปัญหาสำหรับคนที่ชอบสิ่งนั้นเลยครับ
ผมทำในสิ่งที่ตัวเองชอบ ก็อยากทำเเละเรียนรู้ให้มากขึ้น จึงศึกษาตั้งเเต่ตอนนั้นเป็นต้นมา
